# Git və GitHub Praktiki Tapşırıq

Bu repository **Git & GitHub – Branching və Workflow** mövzusunda olan praktiki tapşırığı yerinə yetirmək üçün yaradılmışdır.

## Layihə Haqqında Məlumat
Layihənin məqsədi Git əmrlərindən istifadə edərək branch-lar (budaqlar) yaratmaq, onları idarə etmək və sonda ana (`main`) branch ilə birləşdirmək (merge) bacarığını nümayiş etdirməkdir.

### Görülən işlər:
* Yeni repository yaradıldı və local-a clone edildi.
* `feature-1` branch-ı yaradılaraq README faylı yeniləndi.
* `feature-2` branch-ı yaradılaraq `notes.txt` faylı əlavə olundu.
* Bütün branch-lar `main` branch-ına uğurla merge edildi.
* `fetch` və `pull` əmrləri test edildi.
